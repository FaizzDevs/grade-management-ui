# Design Decisions & Architecture

## 1. Arsitektur
- **Next.js** digunakan untuk routing karena mendukung server component dan optimal untuk UI dashboard.
- **State Management** menggunakan Zustand untuk progress input nilai mahasiswa.
- **UI** menggunakan kombinasi Material UI dan Tailwind CSS untuk fleksibilitas dan konsistensi tampilan.

## 2. UX Focus
- Dashboard dirancang **intuitif untuk dosen**, dengan informasi ringkas:
  - Jumlah kelas & mahasiswa.
  - Progress pengisian nilai.
- Akses cepat ke:
  - Input Nilai.
  - Konfigurasi nilai.

## 3. AI Collaboration
AI digunakan sebagai asisten untuk membantu brainstorming, memberikan saran desain, dan mempercepat pembuatan boilerplate kode.  
Semua implementasi dan pengujian dilakukan oleh developer untuk memastikan kualitas dan kesesuaian dengan kebutuhan.
